Simple HTTP Servlet Library

Depends on modules: http_common, common

 

Reference card (typical usage patterns)

Typical HTTP connection: receiving request and sending response

This library is modeled similarly to Servlets but much much simpler and gives full access to underlying HTTP request/response.

First create a servlet that extends class HttpMiniServlet and implements one abstract method service(), for example:

public class XnServlet extends HttpMiniServlet {
    public void service(HttpServerRequest req, HttpServerResponse resp) throws HttpServerException
    {
    }
}

To have working service it is necessary to plug your  into mini HTTP server to use it:

        HttpMiniServer server = new HttpMiniServer(port);
        XnServlet servlet = new XnServlet();
        server.useServlet(servlet);
        server.startServer();

Now examine interfaces HttpServerRequest and HttpServerResponse to see what methods are available.

Demonstrate different ways to implement Web Service by using WS/XSUL.

Default HTTP container supports hosting both RPC and XML message based service implementations.

The container allows to add global and service specific handlers.

Regardless if service is RPC or MSG based when Java class implementaiton is hosted in HTTP container then the container automatically takes care of WS-Addressing and SOAP-RPC invocations (for free supoprts asynchronous invocations and wsa:ReplyTo/FaultTo).

Virtual security handler that is injected in server processing to provide capability based authz.

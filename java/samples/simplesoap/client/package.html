Demonstrate different ways to interact with Web Services using RPC and Message style interfaces.

Dynamic Invocations

Use XsulDynamicInvoker to invoke WSDL directly (no code or compilation required).

Simple RPC

InteropRpc.java

Use xwsdlc to generate Java interface that is used to interact with Web Serice - message parts are mapped to Java types.

RPC with complext types

WeatherRpc.java

Message style

WeatherMsg.java

Use xwsdlc to generate Java interface (--type=message) that is used to interact with Web Serice - whole XML message can be manipullated.
